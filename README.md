## Sobre o projeto

**Crud completo, com criação, leitura, atualização e remoção**

### Como instalar:
**Baixe o apk no seu smartphone, cerifique-se de que seu smartphone está configurado para aceitar apps de terceiros, assim não terá conflitos.**
